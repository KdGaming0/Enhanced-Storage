## Update 1.0.0-beta.1

- Full rewrite of the codebase to improve reliability and maintainability.
- Added new features including the option to edit the title of any page and the position the page will appear in. So you can move page 1 to be the 5th page in the list instead of it being the first, or change its title from Ender Chest 1 to Weapons. To do so just Right Click the page.
- Added new buttons to the menu on the right side of the inventory to allow for quick access to the settings menu and to change the theme of the menu. (These can be disabled in the settings menu if you don't want them there.)
- Some other minor bug fixes and improvements to the user interface.
